Таблица размеров
Размеры
Для того чтобы правильно определить свой размер одежды, важно грамотно снять мерки. Лучше всего измерять показатели поверх нижнего белья, которое Вы обычно носите.
Обхват груди
Необходимо, чтобы спереди сантиметровая лента проходила по наиболее выступающим точкам груди, сбоку — по подмышечным впадинам, на спине — немного выше.
Обхват под грудью
Необходимо выдохнуть воздух, приложить сантиметр прямо под грудь, так чтобы он достаточно плотно прилегал к телу. Сантиметр удерживаете параллельно полу.
Обхват талии
Измеряется строго горизонтально по естественной линии талии.
Обхват бедер
Сантиметровая лента должна проходить строго горизонтально, по наиболее выступающим точкам ягодиц.
Таблицы размеров
Измерения будут намного точнее, если учитывать несколько хитростей:
—
Для снятия мерок лучше воспользоваться помощью второго человека
—
Не стоит пользоваться старой сантиметровой лентой
—
Лента должна прилегать плотно к телу по измеряемой линии, при этом нельзя допускать ни ее провисания, ни затягивания
—
Стойте без напряжения, сохраняя привычную осанку
